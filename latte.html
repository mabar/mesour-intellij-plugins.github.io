IntelliJ Latte Pro

Work in progress...

Provides support for Latte – a template engine for PHP.

This is faster and more advanced plugin than open-source Latte plugin. It's a must-have plugin for all professional Nette developers.

Current status

  • Development started in Kotlin instead of Java
  • jFlex lexer and BNF parser is currently developing

Planed features

  • Everything what have open-source plugin but really faster
  • No problems with big projects
  • Completion and references for included templates (files)
  • Completion and type check for method/function parameter
  • Completion for links and goto from presenter to template
  • Support for generic types
